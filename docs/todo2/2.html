認証機能その２

認証機能では、ログインに成功すると/homeのURLへリダイレクトするようになっています。
これをカスタマイズするには、app/Http/Controllers/Auth 配下の LoginController、RegisterController、ResetPasswordController、 VerificationControllerそれぞれのredirectToプロパティで、リダイレクト先の場所を設定することで変更できます。

class LoginController extends Controller
{
    /*
    |--------------------------------------------------------------------------
    | Login Controller
    |--------------------------------------------------------------------------
    |
    | This controller handles authenticating users for the application and
    | redirecting them to your home screen. The controller uses a trait
    | to conveniently provide its functionality to your applications.
    |
    */

    use AuthenticatesUsers;

    /**
     * Where to redirect users after login.
     *
     * @var string
     */
    protected $redirectTo = '/home';

今回はユーザー登録とログインした後には、タスク一覧に遷移させたいのでLoginController、RegisterControllerを 下記のように変更してしまいましょう。

    protected $redirectTo = '/task';

ユーザー登録に関するバリデーション等のカスタマイズ

新しいユーザーを登録する場合に入力してもらうフォームのフィールドを変更したり、 データベースに新しいユーザーレコードを登録する方法をカスタマイズしたい場合は、 RegisterControllerクラスを変更することができます。

RegisterController の validatorメソッド には新規ユーザーに対するバリデーションルールを定義してあります。 このメソッドは自由に内容を変更することができます。

    /**
     * Get a validator for an incoming registration request.
     *
     * @param  array  $data
     * @return \Illuminate\Contracts\Validation\Validator
     */
    protected function validator(array $data)
    {
        return Validator::make($data, [
            'name' => ['required', 'string', 'max:255'],
            'email' => ['required', 'string', 'email', 'max:255', 'unique:users'],
            'password' => ['required', 'string', 'min:8', 'confirmed'],
        ]);
    }

また、RegisterController の createメソッド は新しいApp\Userモデルのレコードを、Eloquentを使ってデータベースに登録します。 必要に合わせて自由に内容を変更することができます。

    /**
     * Create a new user instance after a valid registration.
     *
     * @param  array  $data
     * @return \App\User
     */
    protected function create(array $data)
    {
        return User::create([
            'name' => $data['name'],
            'email' => $data['email'],
            'password' => Hash::make($data['password']),
        ]);
    }

認証済みユーザーの取得

ログイン済みのユーザー情報を取得したい場合は、Authファサードを使使うことができます。

use Illuminate\Support\Facades\Auth;

// 現在認証されているユーザーの取得
$user = Auth::user();

// 現在認証されているユーザーのID取得
$id = Auth::id();

    }
