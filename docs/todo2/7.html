総復習

これまで学習した内容のおさらいをしながらTODOアプリの改修を行います。

課題１

TodoController から return で呼び出される view をすべて resources/views/task の中に移動して 移動先のファイルを view で呼び出すように修正しましょう。


課題２

UserControllerを作成して、登録されているユーザー専用のプロフィールページを作成してください。 このとき、プロフィールはログインしているユーザーが自分のものだけを登録／更新出来るようにしましょう。
テーブルについては、マイグレーションを使って新規で user_prifiles を作成してください。
カラムとして user_id を必ず持つように作成しましょう。
