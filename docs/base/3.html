ビュー（View）の作成

ビュー（View）

https://readouble.com/laravel/5.8/ja/views.html

ビューとはMVCモデルのVに該当します。 簡単に説明するとHTMLを書く部分で、見た目の部分を担っています。
Laravelでは、Blade というテンプレートエンジンと呼ばれるものが採用されていて、 コントローラから変数が渡された場合、それをビューでも利用することができます。

ビューの作成

早速、Viewを作成して見ましょう。
Laravelプロジェクトフォルダのapp/resources/views配下にサンプルのファイルを作成します。 ここでは、index.blade.php という名前として解説を進めていきます。
Bladeテンプレートエンジンで使用するファイルの拡張子は.blade.phpです。.php だけでは無いことに注意しましょう。

作成した index.blade.php にHTMLを記述して表示させてみます。index.blade.php に下記の内容を記述してください。

<html>
    <body>
        <h1>{{ $title }}ページ</h1>
        これは最初のViewです。<br>
    </body>
</html>

次に、ブラウザからアクセスされた際にこの index.blade.php を表示するよう、コントローラを修正します。 TodoController.phpのindexメソッドを次のように修正してください。

    public function index ()
    {
        return view('index', [
            'title' => 'サンプル'
        ]);
    }

作成と修正が終わったら、実際にブラウザからhttp://localhost:8000にアクセスして動作を確認してみましょう。

この様にLaravelが持つviewというメソッドが index.blade.php を呼び出し、 その際に title という変数に「サンプル」という文字を代入した形で付与することで、 コントローラで宣言した変数をビューで表現することが可能になります。

view('読み込むViewの.blade.phpを削除した名前', 変数と値を紐付ける配列);

// ▼ 変数と値を紐付ける配列の書き方
// [ '変数名' => 変数に代入する値 ]

ビューで扱うことの出来る変数

上記で説明しているようにビューでは、コントローラ〜受け取った変数名と値を受け取ることで、 HTML内にコントローラで加工した値を利用して行くことができます。
