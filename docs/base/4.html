Bladeの書き方

Bladeの書き方

https://readouble.com/laravel/5.8/ja/blade.html

Laravelでビューを構成しているのは、Bladeと呼ばれるテンプレートエンジンだという説明を先に書きました。 それでは、Bladeとはどういうものなのでしょうか？実際の書き方を例に説明していきたいと思います。

HTML内でBladeを利用する場合、書き方に従って変数などを記述するだけでecho処理されるため、 それだけでその内容をブラウザに表示することができます。また、この書き方には２つ存在します。

【エスケープあり】
これは表示されるとき、PHPのhtmlspecialchars関数を通します。

{{ 内容 }}

【エスケープなし】
これは表示されるとき、PHPのhtmlspecialchars関数を通しません。

{!! 内容 !!}

htmlspecialchars
https://www.php.net/manual/ja/function.htmlspecialchars.php

Bladeの制御構文

それではBladeの制御構文を説明していきます。 制御構文とは、HTML内でBladeをプログラム的に利用するための書き方と考えてください。

If文

条件式をつけて表示内容を分岐することができます。

@if (条件式)
    if条件に通過した場合の処理
@elseif (条件式)
    else if条件に通過した場合の処理
@else
    if にも elseif にも条件一致しなかった場合の処理
@endif

また、if ( ! 条件式) をスマートに表現可能にした unless ディレクティブもあります。

@unless ($is_login)
    あなたはログインしていません。
@endunless

変数の存在有無を確認したい場合は、PHPのように isset や empty も用意されています。

@isset($variable)
    isset に通過した場合の処理
@endisset

@empty($variable)
    empty に通過した場合の処理
@endempty

Switch文

１つの変数に対して複数の選択肢が存在する場合は switch 構文を利用することで分岐をもたせることができます。

@switch($i)
    @case(1)
        最初のケース
        @break
    @case('ほげ')
        ２番めのケース
        @break
    @default
        デフォルトのケース
@endswitch

繰り返し

ループ処理が必要な場合、必要に応じた様々な繰り返し処理をすることもできます。 これは例えば、配列に格納された複数のデータを同じフォーマットで表示する場合などにも有効です。

@for ($i = 0; $i < 10; $i++)
    現在の値は： {{ $i }}
@endfor

@foreach ($users as $user)
    <p>これは {{ $user->id }} ユーザーです。</p>
@endforeach

@forelse ($users as $user)
    <li>{{ $user->name }}</li>
@empty
    <!-- $users配列が空の場合 -->
    <p>ユーザーなし</p>
@endforelse

@while (true)
    <p>無限ループ中</p>
@endwhile

【繰り返し処理のスキップ】
条件次第で、繰り返しを修了したり、現在の処理をスキップさせたい場合もあると思います。 その場合は、PHPと同様に continue や break を挟むことができます。

@foreach ($users as $user)
    @if ($user->id == 1)
        <!-- 現在をスキップ -->
        @continue
    @endif
    <li>{{ $user->name }}</li>
    @if ($user->id == 5)
        <!-- 繰り返しの終了 -->
        @break
    @endif
@endforeach

また、もっとシンプルな方法として continue や break に直接条件を指定する方法も利用できます。

@foreach ($users as $user)
    @continue($user->id == 1)
    <li>{{ $user->name }}</li>
    @break($user->id == 5)
@endforeach

【ループ変数】
繰り返しを行っている処理の間では、ループ変数（$loop）を使うことで現在のループインデックスや現在の繰り返し状況などを知ることができます。 $loopという変数は、繰り返し処理が開始した時点で自動的に割り当てられ、繰り返し処理が修了した時点で自動的に破棄されます。

@foreach ($users as $user)
    @if ($loop->first)
        これは最初の繰り返し
    @endif

    @if ($loop->last)
        これは最後の繰り返し
    @endif

    <p>これは {{ $user->id }} ユーザーです。</p>
@endforeach

入れ子状態のループでは、parentプロパティを通じて現在の親の状況を知ることもできます。

@foreach ($users as $user)
    @foreach ($user->posts as $post)
        @if ($loop->parent->first)
            これは親のループの最初の繰り返しだ
        @endif
    @endforeach
@endforeach

$loopの便利なプロパティ

プロパティ 意味
$loop->index 現在のループインデックス
$loop->iteration 現在の繰り返し数
$loop->remaining 繰り返しの残数
$loop->count 繰り返している配列の要素数
$loop->first 最初の繰り返しかどうか（TRUE or FALSE）
$loop->last 最後の繰り返しかどうか（TRUE or FALSE）
$loop->even 偶数回目の繰り返しかどうか（TRUE or FALSE）
$loop->odd 奇数回目の繰り返しかどうか（TRUE or FALSE）
$loop->depth 現在のループのネストレベル（何階層目の入れ子か）
$loop->parent ネスト状態の場合の親の$loop変数そのもの

コメントアウト

コメントアウトも可能です。コメントアウトした場合、Bladeソースとしては残りますがブラウザ上には表示されません。

{{-- このコメントはHTMLには表示されない --}}

PHPの記述

{{ }}内にPHPの関数を記述することで、PHPと通して変数などを表示することもできますが、PHPとしての処理をBlade内に記述することもできます。

@php
    // これはPHPとしての処理
    $var = 123;
    if ($var === 234) {
        echo '変数は' . $var;
    } else {
        echo '変数は234じゃありませんでした。';
    }
@endphp

また、phpディレクティブを利用して宣言された変数は、Blade内で利用することもできます。

@php
    // これはPHPとしての処理
    $var = 123;
@endphp

<p>{{ $var }}</p>
