タスク編集画面の作成

タスク編集画面の作成

ルーティングとしては下記のURLを実行した場合の処理を実装していきます。

// タスク編集画面
Route::get('/edit/{id}', 'TodoController@edit')->name('task.eidt');

ToDoControllerクラスのcreateメソッドにルーティングが設定されているのでTodoController.phpを編集していきます。
まずは、editメソッドを追加しましょう。

    public function edit ()
    {

    }

ここにタスク更新のための処理を書いていくのですが、今回は新規作成と違い既に登録されているタスクを取得する必要があります。 ルーティングとしてタスクIDを設定するようにしていますので、そのIDから該当のタスクを特定することにします。
そのままコントローラで取得するという方法もありますが、今回はあえてモデルを利用していきたいと思います。 下記のようにTaskモデル（Task.php）にメソッドを追加してみましょう。

    public function get_task_by_id($id)
    {
        return $this->where('id', $id)->first();
    }

そして追加したメソッドをコントローラーから呼び出してViewに受け渡します。

    public function edit ($id)
    {
        $tasks = new Task;
        $task_data = $tasks->get_task_by_id($id);
        return view('edit', [
            'task' => $task_data,
        ]);
    }

次にviewのテンプレートを作成していきます。
作りとしては、create.blade.php を流用する形が良いでしょう。

<!DOCTYPE html>
<html>
<head>
    <meta charset="utf-8">
    <meta http-equiv="X-UA-Compatible" content="IE=edge">
    <title>タスク一覧</title>
    <link rel="stylesheet" href="">
</head>
<body>
    <form action="{{route('task.update', ['id' => $task->id])}}" method="POST">
        @csrf
        タスク名：<input type="text" name="name" value="{{ $task->name }}" placeholder=""><br>
        @if ($errors->has('name') )
        <span>{{ $errors->first('name') }}</span><br>
        @endif
        期日　　：<input type="date" name="deadline" value="{{ $task->deadline }}" placeholder=""><br>
        @if ($errors->has('deadline') )
        <span>{{ $errors->first('deadline') }}</span><br>
        @endif
        詳細　　：<textarea name="detail">{{ $task->detail }}</textarea><br>
        @if ($errors->has('detail') )
        <span>{{ $errors->first('detail') }}</span><br>
        @endif
        <input type="submit" value="編集">
    </form>
</body>
</html>

ここでは、データベースから取得した値を表示のデフォルト値として表示するようにしています。 しかし、ブラウザで確認すると解ると思うのですが、「期日」については表示がされません。
これは、<input type="date" ... で使用されるフォーマットとデータベースから取得した値に 相違があるからです。

# input type="date" の値
2020/01/01

# データベースの値
2020-03-16 00:00:00

ミューテタという仕組みを使うことでこれを解決する事ができます。
