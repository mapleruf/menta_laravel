タスク一覧画面の作成

タスク一覧画面の作成

ルーティングとしては下記のURLを実行した場合の処理を実装していきます。

// タスク一覧画面
Route::get('/', 'TodoController@index');

このルーティングでは、ToDoControllerクラスのindexメソッドにルーティングが設定されているので TodoController.phpを編集していきます。
また、ここではTaskモデルをuseしているということを思い出しておいてください。

    public function index ()
    {
        $task = new Task;
        $list = $task->get_task();

        return view('index', [
            'title' => 'サンプル',
               'list'  => $list
        ]);
    }

上記のindexメソッドを下記のように書き換えましょう。

    public function index ()
    {
        $task = new Task;
        $list = $task->get_task();

        return view('index', [
               'list'  => $list
        ]);
    }

また、viewメソッドで呼び出しされているindex.blade.phpも一緒に編集します。

<!DOCTYPE html>
<html>
<head>
    <meta charset="utf-8">
    <meta http-equiv="X-UA-Compatible" content="IE=edge">
    <title>タスク一覧</title>
    <link rel="stylesheet" href="">
</head>
<body>
    <div>
        <a href="{{ route('task.create') }}">タスクの追加</a>
    </div>
    <table>
        <tr>
            <th>タスク名</th>
            <th>詳細</th>
            <th>期日</th>
        </tr>
        @foreach ($list as $task)
        <tr>
            <td>{{ $task->name }}</td>
            <td>{{ $task->detail }}</td>
            <td>{{ $task->deadline }}</td>
        </tr>
        @endforeach
    </table>
</body>
</html>

各ファイルの編集ができたら、ブラウザからhttp://localhost:8000にアクセスして動作を確認してみましょう。

ここで route() という関数が出てきているのに気がついたでしょうか？これはヘルパ関数と呼ばれるLaravelの便利機能になります。 route('ルーティング名')という書き方をすることで、ルーティングされているそのページへのURLを生成することができます。

上記の場合、ルーティングで name('task.create') としたルーティングのURLを生成しています。

// タスク新規作成画面
Route::get('create', 'TodoController@create')->name('task.create');
