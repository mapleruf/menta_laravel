タスク更新処理の作成

タスク編集画面の作成が完了したら、早速タスクの登録処理を実装していきます。

// タスク更新処理
Route::post('/update/{id}', 'TodoController@update')->name('task.update');

上記の通り、ToDoControllerクラスのupdateメソッドにルーティングが設定されているのでTodoController.phpを編集していきます。
まずは、updateメソッドを追加しましょう。

    public function update (Request $request, $id)
    {

    }

そしてデータベースのデータを更新する処理を作っていきます。

    public function update (Request $request, $id)
    {
        $input = $request->all();
        unset($input['_token']);

        $tasks = new Task;
        $tasks = $tasks->find($id);
        $tasks->fill($input);
        $tasks->save();

        return redirect(route('task.index'));
    }

ここでポイントとなるのは、create メソッドと同じく save を使っているという点です。 モデルの持つ save メソッドは、既存のモデルデータうを利用すると UPDATE、それなければ INSERT と 切り替えて実行することができます。
そのため、UPDATE を実行するには、更新対象のデータを一度取得する必要があります。

        $tasks = new Task;
        $tasks = $tasks->find($id);

この書き方は少し奇妙に見えるかも知れませんね。
Laravelのモデルによってデータを取得した場合、特定のケースを除いて下記のようなモデルオブジェクトというものが返却されます。

App\Task Object
(
    [fillable:protected] => Array
        (
            [0] => name
            [1] => detail
            [2] => deadline
        )

    [connection:protected] => mysql
    [table:protected] => tasks
    [primaryKey:protected] => id


... （省略） ...

)

この情報を元に UPDATE を実行するのです。

最後に index.blade.php に編集用のリンクを追加します。

<!DOCTYPE html>
<html>
<head>
    <meta charset="utf-8">
    <meta http-equiv="X-UA-Compatible" content="IE=edge">
    <title>タスク一覧</title>
    <link rel="stylesheet" href="">
</head>
<body>
    <div>
        <a href="{{ route('task.create') }}">タスクの追加</a>
    </div>
    <table>
        <tr>
            <th>タスク名</th>
            <th>詳細</th>
            <th>期日</th>
        </tr>
        @foreach ($list as $task)
        <tr>
            <td>{{ $task->name }}</td>
            <td>{{ $task->detail }}</td>
            <td>{{ $task->deadline }}</td>
            {{-- リンクを追加 --}}
            <td><a href="{{ route('task.edit', ['id' => $task->id]) }}">編集</a></td>
        </tr>
        @endforeach
    </table>
</body>
</html>

修正が終わったら早速、タスク編集画面から更新処理をしてみましょう。
